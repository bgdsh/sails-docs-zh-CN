# myApp/api/controllers/UserController.js
### 用途

当你运行 'sails generate api User'的时候，这个文件被创建。它包含了'User'模型的所有控制器逻辑。
这是你存放『控制器动作』的地方。控制器动作是用来给客户端发送数据、渲染展示数据的页面的。
